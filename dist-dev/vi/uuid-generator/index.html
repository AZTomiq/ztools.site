UUID Versions Explained

  • Version 4 (Random): Generated using random numbers. The most common and recommended widely.
  • Version 1 (Time-based): Uses timestamp and random node (since MAC is private). Uniqueness guaranteed if generated at different times.
  • Version 7 (Time-ordered): Newer standard. Sortable by time (starts with timestamp). Great for database keys.
