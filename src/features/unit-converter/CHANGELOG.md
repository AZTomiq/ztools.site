# Changelog - Quy đổi đơn vị

## [1.1.0] - 2025-12-18
### Added
- Thêm tab **Dung lượng (Storage)** hỗ trợ các đơn vị máy tính (Bit, Byte, KB, MB, GB, TB, PB).
- Hợp nhất các đơn vị **Hệ Mỹ (US Imperial)** vào tab Thể tích (fl oz, cup, pint, quart, tsp, tbsp).

## [1.0.0] - 2025-12-17
### Added
- Phiên bản đầu tiên với các đơn vị cơ bản: Chiều dài, Khối lượng, Thể tích, Diện tích, Tốc độ, Nhiệt độ.
- Tính năng hiển thị công thức và bảng so sánh nhanh.
